🚀 How to Run
Save the game code to a Python file, e.g. snake_game.py

Run the game with:

bash
Copy
Edit
python snake_game.py
🎮 Controls
Action	Key
Move Up	↑ or W
Move Down	↓ or S
Move Left	← or A
Move Right	→ or D
Restart Game	R (after game over)
Quit Game	Q (after game over)
